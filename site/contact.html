Meet the Zeditors

We are all employed by IBM Australia as System Z technical specialists, and share a passion for testing things out on our z/OS test rigs (we use IBM Z Development and Test software to run z/OS systems as a virtualized guest of Linux on some beefy x64 systems at home).

As our recipe books for getting things done grew larger, we wanted a permenant place to safely store them and share them. Github was the obvious repository for our documentation and code, and Github Pages was the equally obvious means to publish it. This website is the current state of play.

Everything on this site is free to read and download anonymously.
