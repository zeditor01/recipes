Environment for CDC Worked Examples.

These CDC Worked Examples are based upon the systems landscape illustrated below

IBM InfoSphere Data Replication ( aka "CDC" ) provides a data replication capability between lots of heterogeneous data sources.

  • Some data sources ( like IMS and VSAM ) can only act as CDC sources CDC Cap.
  • Some data sources ( like Apache Kafka ) can only act as CDC targets CDC App.
  • Some data sources ( like DB2 z/OS ) can act as both CDC sources and CDC targets.
  • A set of common tools operate with all CDC agents.
  • CDC integrates with industry standards to support common authentication and encryption services.

    A CDC Source can replicate data to any CDC Target

    A CDC Target can receive replicated data from and CDC Source
